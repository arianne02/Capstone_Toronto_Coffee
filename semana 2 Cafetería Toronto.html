Batalla de los Vecindarios¶

Introducción¶

En este documento se van a plantear diferentes aspectos a tener en cuenta a la hora de escoger el sitio ideal para una cafetera en la ciudad de Toronto. Se analizarn los diferentes vecindarios de Toronto en busca de informacin cmo nmero de cafeteras en la zona, densidad de poblacin, que sirvan como apoyo para tomar estas decisiones. El objetivo del proyecto es identificar que vecindarios donde abrir una cafetera podra tener mayor potencial de xito, lo que implica buscar reas con alta afluencia de personas, pero con pocas cafeteras existentes. Un anlisis de este tipo es valioso para empresas y emprendedores permitindoles minimizar la competencia y maximizar la visibilidad y clientela potencial.

def getLocationData(neighborhood, postal_code):
    # Headers y parmetros para la solicitud
    headers = {
        "Accept": "application/json",
        "Authorization": API_KEY
    }
    params = {
        "ll": neighborhood, #'40.7128,-74.0060'
        "radius": RADIUS,
        "query": QUERY,
        "limit": LIMIT
    }

    # Llamada a la API de Foursquare Places v3
    url = "https://api.foursquare.com/v3/places/search"
    response = requests.get(url, headers=headers, params=params)

    # Verifica que la respuesta sea exitosa
    if response.status_code == 200:
        data = response.json()
        # Procesa y organiza los datos de los lugares
        cafeterias = []
        for place in data['results']:
            cafeterias.append({
                'Nombre': place['name'],
                'Latitud': place['geocodes']['main']['latitude'],
                'Longitud': place['geocodes']['main']['longitude'],
                'Direccin': place['location'].get('formatted_address', 'No especificado'),
                'Vecindario': place['location'].get('neighborhood', ['No especificado'])[0]
            })

        # Crear un DataFrame con los datos obtenidos
        df_cafeterias = pd.DataFrame(cafeterias)
        ## Filtramos que coincida el cdigo postal
        df_cafeterias['CdigoPostal'] = df_cafeterias['Direccin'].apply(lambda x : x[-7:-4])
        df_cafeterias = df_cafeterias[df_cafeterias['CdigoPostal'] == postal_code]
        return df_cafeterias
    else:
        print("Error en la solicitud:", response.status_code, response.text)
        print("datos:" +  postal_code)
        return []
